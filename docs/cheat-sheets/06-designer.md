# DESIGNER (Nhân viên Thiết kế) — Cheat Sheet

> 🔑 Đăng nhập: `designer@jamahome.vn` / `designer123`

## Bạn thấy gì trên Sidebar

```
✅ Dashboard    ❌ Pipeline    ✅ Dự án     ✅ Khách hàng
❌ Hợp đồng    ✅ Báo giá     ❌ Kho       ❌ Kế toán
❌ Nhân sự     ❌ Tài chính   ❌ P&L       ❌ Báo cáo    ✅ Cài đặt
```

## Việc cần làm hàng ngày

| Thời gian | Việc | Trang |
|-----------|------|-------|
| Sáng | Xem Dashboard cá nhân | `/` |
| Cả ngày | Làm thiết kế theo đầu việc | `/projects` |
| Khi xong | Cập nhật trạng thái task | Click task → "Done" |
| Khi xong | Upload file bản vẽ | Click task → Upload file |
| Cuối tuần | Tổng kết tiến độ | `/projects` |

## Quyền chính

| Việc | How |
|------|-----|
| Xem dự án được giao | `/projects` |
| Cập nhật trạng thái task | Click task → chọn Done/In Progress |
| Upload file bản vẽ | Click task → Upload file hoặc dán link |
| Ghi chú tiến độ | Click task → nhập ghi chú → "Gửi" |
| Tạo báo giá thiết kế | `/quotations` → "+ Báo giá mới" |

## Các task thiết kế bạn phụ trách

| Stage | Task | Output |
|-------|------|--------|
| Thiết kế | 2D Concept | File concept phẳng |
| Thiết kế | 3D Demo | Model 3D demo |
| Thiết kế | 3D Render | Render chất lượng cao |
| Thiết kế | Technical Drawing | Bản vẽ kỹ thuật |
| Thiết kế | Final Design | Gói thiết kế hoàn chỉnh |

## Mẹo

- Upload **file kết quả chốt** (Final File) để PM/Khách easily truy cập
- Ghi chú rõ ràng trên mỗi task để team theo dõi
- Dùng **Google Drive** → dán link vào "File kết quả chốt"

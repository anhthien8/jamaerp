# ADMIN (Giám đốc) — Cheat Sheet

> 🔑 Đăng nhập: `admin@jamahome.vn` / `admin123`

## Bạn thấy gì trên Sidebar

```
✅ Dashboard    ✅ Pipeline    ✅ Dự án     ✅ Khách hàng
✅ Hợp đồng    ✅ Báo giá     ✅ Kho       ✅ Kế toán
✅ Nhân sự     ✅ Tài chính   ✅ P&L       ✅ Báo cáo    ✅ Cài đặt
```

## Việc cần làm hàng ngày

| Thời gian | Việc | Trang |
|-----------|------|-------|
| Sáng | Xem Dashboard tổng quan | `/` |
| Sáng | Kiểm tra leads quá hạn | `/leads` |
| Trưa | Duyệt báo giá chờ | `/quotations` |
| Chiều | Kiểm tra tiến độ dự án | `/projects` |
| Cuối tuần | Xem P&L, báo cáo tài chính | `/pl` |

## Quyền đặc biệt (chỉ Admin mới có)

| Việc | How |
|------|-----|
| Thêm/sửa/xóa nhân viên | `/hr` → "+ Thêm Nhân viên" |
| Vô hiệu hóa tài khoản | `/hr` → chọn NV → "Nghỉ việc" |
| Quản lý bậc lương | `/finance` → Tab "Bậc lương" |
| Quản lý chi phí cố định | `/finance` → Tab "Định phí" |
| Xem P&L toàn công ty | `/pl` |
| Duyệt vật tư vượt định mức | `/inventory` hoặc Telegram |

## Phím tắt

| Phím | Chức năng |
|------|-----------|
| `Ctrl + K` | Tìm kiếm toàn cục |
| `Esc` | Đóng popup |

## Mẹo

- Chuyển **Demo Mode** → **Work Mode** bằng cách bấm toggle ở Sidebar
- Dùng **Telegram Bot** để xem pipeline khi đi ngoài

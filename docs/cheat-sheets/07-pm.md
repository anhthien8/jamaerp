## Quyền chính

| Việc | How |
|------|-----|
| Quản lý toàn bộ task dự án | `/projects` → click dự án → dropdown status inline |
| Tạo dự án mới | `/projects` → "+ Dự án mới" |
| Bulk assign tasks | API: POST /projects/{id}/tasks/bulk-assign |
| Gửi yêu cầu vật tư | Click dự án → "📦 Yêu cầu vật tư" |
| Upload ảnh thi công | Click task → 📷 Đính kèm ảnh (chọn nhiều) |
| Xem lịch sử khách hàng | `/customers` → click khách → xem danh sách dự án |
| Xem kho vật tư | `/inventory` |
| Upload file + ảnh nghiệm thu | Click task → Ghi chú + Đính kèm ảnh |
| Xem lịch sử dự án của khách | `/customers` → click khách → xem danh sách dự án |

## Quy trình quản lý dự án

```
1. Lead ký HĐ → Hệ thống tạo dự án + 19 tasks (tự động)
2. Bạn review tasks → Phân công cho designer/thu mua
3. Theo dõi tiến độ trên Kanban (5 giai đoạn)
4. Khi cần vật tư → Gửi yêu cầu qua hệ thống
5. Thu mua duyệt → Vật tư về → Cập nhật task
6. Upload ảnh + ghi chú thi công tại từng task
7. Nghiệm thu → Upload biên bản → Hoàn thành
```

## Khách hàng đa dự án

Một khách có thể có nhiều dự án. Để xem:
1. Vào `/customers` → click tên khách
2. Modal hiện ra → cuộn xuống xem **"📋 Dự án"**
3. Thấy danh sách tất cả dự án + tiến độ + giá trị

## Yêu cầu vật tư nhanh

1. Click vào dự án trên Kanban
2. Click nút **"📦 Yêu cầu vật tư"**
3. Nhập tên vật tư, số lượng, đơn vị
4. Hệ thống tự gửi đến **Thu mua** duyệt
5. Nhận thông báo khi được duyệt/từ chối

## Upload ảnh thi công (UX mới)

- Click vào task → Gõ ghi chú → Bấm **"📷 Đính kèm ảnh"**
- Có thể chọn **nhiều ảnh cùng lúc** (giữ Ctrl/Shift)
- Xem thumbnails preview trước khi đăng
- Đánh dấu **×** để xóa ảnh không cần
- Counter badge hiện số ảnh đã chọn

## Mẹo

- Dùng **Kanban view** để thấy tổng quan tất cả dự án
- Click vào task → **Ghi chú + Upload ảnh** để lưu bằng chứng thi công
- Đánh dấu task **Done** → Thanh tiến độ tự cập nhật (thứ tự tasks giữ nguyên)
- Xem lịch sử khách hàng trước khi bắt đầu dự án mới

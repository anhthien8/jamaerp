# PM (Trưởng Dự án) — Cheat Sheet

> 🔑 Đăng nhập: `pm@jamahome.vn` / `pm123`

## Bạn thấy gì trên Sidebar

```
✅ Dashboard    ❌ Pipeline    ✅ Dự án     ✅ Khách hàng
✅ Hợp đồng    ✅ Báo giá     ✅ Kho       ❌ Kế toán
❌ Nhân sự     ❌ Tài chính   ❌ P&L       ✅ Báo cáo    ✅ Cài đặt
```

## Việc cần làm hàng ngày

| Thời gian | Việc | Trang |
|-----------|------|-------|
| Sáng | Xem Dashboard team | `/` |
| Sáng | Kiểm tra task trễ deadline | `/projects` |
| Cả ngày | Quản lý tiến độ dự án | `/projects` |
| Khi cần | Gửi yêu cầu vật tư | Click dự án → "Yêu cầu vật tư" |
| Chiều | Cập nhật trạng thái task | Click task → chọn status |

## Quyền chính

| Việc | How |
|------|-----|
| Quản lý toàn bộ task dự án | `/projects` → click dự án |
| Tạo dự án mới | `/projects` → "+ Dự án mới" |
| Gửi yêu cầu vật tư | Click dự án → "📦 Yêu cầu vật tư" |
| Xem kho vật tư | `/inventory` |
| Upload file nghiệm thu | Click task → Upload file |

## Quy trình quản lý dự án

```
1. Lead ký HĐ → Hệ thống tạo dự án + 19 tasks
2. Bạn review tasks → Phân công cho designer/thu mua
3. Theo dõi tiến độ trên Kanban
4. Khi cần vật tư → Gửi yêu cầu qua hệ thống
5. Thu mua duyệt → Vật tư về → Cập nhật task
6. Nghiệm thu → Upload biên bản → Hoàn thành
```

## Yêu cầu vật tư nhanh

1. Click vào dự án trên Kanban
2. Click nút **"📦 Yêu cầu vật tư"**
3. Nhập tên vật tư, số lượng, đơn vị
4. Hệ thống tự gửi đến **Thu mua** duyệt
5. Nhận thông báo khi được duyệt/từ chối

## Mẹo

- Dùng **Kanban view** để thấy tổng quan tất cả dự án
- Click vào task → **Ghi chú + Upload ảnh** để lưu bằng chứng thi công
- Đánh dấu task **Done** → Thanh tiến độ tự cập nhật

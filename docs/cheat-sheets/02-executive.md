# CEO (Ban Quản Trị) — Cheat Sheet

> 🔑 Đăng nhập: `ceo@jamahome.vn` / `ceo123`

## Bạn thấy gì trên Sidebar

```
✅ Dashboard    ❌ Pipeline    ✅ Dự án     ✅ Khách hàng
✅ Hợp đồng    ❌ Báo giá     ❌ Kho       ❌ Kế toán
❌ Nhân sự     ❌ Tài chính   ✅ P&L       ✅ Báo cáo    ✅ Cài đặt
```

## Việc cần làm hàng ngày

| Thời gian | Việc | Trang |
|-----------|------|-------|
| Sáng | Xem Dashboard tổng quan | `/` |
| Thứ 2 | Xem P&L tuần trước | `/pl` |
| Hàng tháng | Xem Báo cáo tổng hợp | `/reports` |
| Khi cần | Xem tiến độ dự án quan trọng | `/projects` |

## Quyền chính

| Việc | How |
|------|-----|
| Xem P&L toàn công ty | `/pl` — lọc theo tháng/quý/năm |
| Xem báo cáo hiệu suất | `/reports` |
| Xem tiến độ dự án | `/projects` |
| Xem hợp đồng | `/contracts` |

## Mẹo

- P&L có thể **sort** theo Doanh thu / Lợi nhuận / Margin
- Dùng bộ lọc **Tháng/Quý/Năm** để so sánh kết quả kinh doanh
- Dự án có margin < 15% sẽ bị **đánh dấu đỏ** cảnh báo
